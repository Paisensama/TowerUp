 Tower Up

One-button stacking game where you drop moving cake layers to build the tallest tower you can. Perfect drops build a combo multiplier. Miss the stack and the tower falls.

 Features
- One-button gameplay with difficulty modes
- Perfect-hit combo multiplier
- Haptic feedback toggle
- Persistent high score (AsyncStorage)
- Dynamic sky themes and smooth animations

 Tech Stack
- Expo + React Native
- Expo Router
- AsyncStorage
